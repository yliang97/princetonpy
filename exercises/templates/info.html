Why Python

Python is one of the most popular programming languages today. It's an easy-to-use scripting language with application in web-development, machine learning, statistical analysis, and much more.

How it works

PrincetonPy contains interactive exercises mirroring those from COS 126. You can complete each exercise and run Python code in your browser. Each exercise is designed to expand and reinforce your knowledge of Python.

Why PrincetonPy

PrincetonPy is catered toward COS 126 graduates. Why spend hours learning fundamentals of programming on CodeAcademy when you can get a comprehensive crash-course in Python, with content from COS 126?

Apply your 126 skills

We have included the Java answers to every exercise as well - you can use your 126 computer science knowledge and apply it to a new language!
